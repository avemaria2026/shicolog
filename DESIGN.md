# シコログ — 設計書

## アプリの目的

回数を「いつ・誰で・何で」と一緒に記録できる、シンプルなプライベートカウンター。
データは端末のブラウザだけに保存され、外部送信は一切しない。

## 仕様の3つの軸

1. **完全プライベート**（データは端末のブラウザのみ。サーバーには一切送られない）
2. **スマホPWA**（ホーム画面に置いてオフラインでも動く）
3. **メタデータあり**（1回ごとに「誰で／何で」を任意で残せる）

## 画面構成（4画面 + 下タブ）

### ① ホーム
- 中央にデカい「＋1」ボタン
- タップ → メモ入力モーダル
- 長押し → メモなしで即記録
- 「今日 / 今月」の合計表示

### ② 履歴
- 新しい順にリスト表示
- タップで編集・削除
- 各レコードに「日時 / 誰で / 何で」

### ③ 統計
- 今月／今週／今日／通算の合計カード
- 月別の棒グラフ（直近6か月）
- 曜日の傾向
- 時間帯の傾向（4時間ごと）
- 「誰で」「何で」ランキング（トップ5）

### ④ 設定
- データ管理（CSV書き出し、全データ削除）
- 端末間同期（引き継ぎコード・JSONファイル）
- セキュリティ（PIN変更、生体認証ON/OFF、ロックON/OFF）
- 外観（端末に合わせる／ライト／ダーク）

## データ仕様

1レコード ＝ 以下のオブジェクト：

```json
{
  "id": "timestamp + random",
  "datetime": "2026-05-23T22:30:00.000Z",
  "who": "自由入力（空欄OK）",
  "how": "プリセット選択 or 自由入力"
}
```

プリセット（「何で」用）：
- インスタ / 動画 / グラビア / 漫画 / その他

「誰で」は過去入力履歴から使用頻度順でチップ候補表示。

## データ保存

- ブラウザの **LocalStorage** に保存（1端末完結）
- アプリ起動時の外部通信は ServiceWorker のキャッシュ取得だけ
- バックアップ：設定からCSV／JSONファイル書き出し
- 端末間移行：引き継ぎコード（gzip+base64文字列）またはJSONファイル

## セキュリティ／プライバシー

- **アプリ起動時に外部送信なし**（分析タグ・トラッキングピクセル・広告SDK全部なし）
- 起動時のロック：PIN（4桁）または生体認証（WebAuthn・指紋/顔）。ロック解除は端末内のみで検証
- バックグラウンド30秒以上で自動再ロック
- 設定でロック無効化も可能（自己責任）
- ServiceWorker はキャッシュのみ、データ送信には使わない

## 広告／収益化

- 表示する広告は **静的なアフィリエイトリンク** のみ（DMM/FANZA等）
- 広告SDK・リアルタイム広告配信は使わない（プライバシー方針維持）
- ユーザーがアフィリエイトリンクをタップしたときだけ外部に遷移
- 設置位置は控えめに（ホーム下部・統計画面下部など）

## 技術スタック

- HTML / CSS / Vanilla JavaScript（依存なし、軽量）
- PWA：`manifest.json` + ServiceWorker
- 圧縮：`CompressionStream` API（端末間引き継ぎコード用）
- 認証：`crypto.subtle`（PINハッシュ）、`WebAuthn`（生体認証）

## 公開ページ（アプリ外）

アドセンスではないが、アフィリエイト導入の信頼性のため、また検索流入の足がかりとして以下を整備：

- トップページ：アプリへの入口、コンセプト紹介
- 使い方ページ：操作の説明、Tips
- プライバシーポリシー：データ取扱方針
- 利用規約：使用条件
- About／お問い合わせ：運営者情報（別名義）

## やらないこと（明確に決まっているNG）

- アカウント登録／ログイン機能
- クラウド同期
- 広告SDK（GoogleAds、Yahoo、Adsterra等のトラッキング型広告）
- データの外部送信（分析・統計サービス含む）
- ユーザーデータの収集

## ロードマップ

- **MVP**：個人版count-appからコードを流用してそのまま稼働（このリリース時点）
- **公開ページ整備**：プライバシーポリシー、利用規約、使い方、About
- **アフィリエイト導線**：DMM/FANZAアフィリエイト承認後、控えめに設置
- **SEO**：Search Console登録、メタタグ整備、サイトマップ送信
- **継続改善**：アクセス見ながら機能追加・UX調整
