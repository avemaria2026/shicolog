# シコログ

回数を「いつ・誰で・何で」と一緒に記録できる、プライベートなカウンターアプリ。

- データは端末のブラウザだけに保存。サーバーには一切送信されません。
- スマホのホーム画面に追加してアプリのように使えます（PWA）。
- PINロックは **任意**（必要なら設定画面から有効化／生体認証併用も可）。
- 端末間の引き継ぎは「引き継ぎコード」または「JSONファイル」で。

## 使い方

1. アプリを開く（そのまますぐ使えます）
2. ホーム画面の「＋1」ボタンを **タップ** → 「誰で／何で」を入れて保存（スキップ可）
3. **長押し** すればメモなしで即記録

詳しい使い方は[アプリ内ヘルプ](/help.html)を参照。

## プライバシー

- アプリ起動時の外部通信はありません（広告SDK・分析タグ・トラッキングなし）
- 表示している広告は **静的なアフィリエイトリンク** のみ。あなたがリンクをタップしたときだけ外部に遷移します。
- 詳細は[プライバシーポリシー](/privacy.html)を参照。

## 技術スタック

- HTML / CSS / Vanilla JavaScript（依存ゼロ）
- PWA（Service Worker + manifest）
- 端末間移行：CompressionStream で gzip 圧縮 + base64
- 認証：WebAuthn（生体認証）、crypto.subtle（PINハッシュ）
