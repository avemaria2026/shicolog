使い方

基本：記録する

  1. ホーム画面の大きな「＋1」ボタンを タップ
  2. 「誰で／ジャンル」を入力 → 「保存」（スキップで未入力のまま記録もOK）
  3. 長押し（0.5秒） で、モーダルを開かずにメモなしで即記録もできます

女優名・ジャンルがパッと思い出せないとき：🎲ボタン

記録モーダルの「誰で？」「ジャンルは？」入力欄の隣にある 🎲 をタップすると、 内蔵のリスト（人気女優70人弱／FANZA人気ジャンル70個前後）から1つがランダムで入力されます。 連打すると振り直し。気に入った値で 🔎 FANZA ボタンを押せば、そのキーワードでFANZAを検索できます。

お気に入り女優を登録する（ホーム画面で活用）

  1. ＋1ボタンをタップして記録モーダルを開く
  2. 「誰で？」の候補チップの横にある ☆ をタップ → ★ に変わって登録完了
  3. ホーム画面の「今日はだれにする？」セクションに並ぶ。タップで FANZA検索が新タブで開く
  4. 「最後に記録してからの日数」が一緒に表示されるので、ご無沙汰の人が一目で分かる
  5. 長押しでお気に入りから外せる

履歴を編集・削除

  1. 下タブ「履歴」を開く
  2. 編集したい記録を タップ
  3. 内容を直して「保存」、または「削除」（確認ダイアログあり）

記録をXにシェア（女優の作品ページにリンク付き）

履歴の各レコード右端の 📤 X ボタンをタップすると、 その記録の女優名・ジャンルを含む文面と、女優のFANZA作品検索リンクが入った投稿画面が開きます。 匿名アカウントで「今日は○○で賢者になりました」みたいに使うと拡散の種になります。 ※ メインの実名Xアカウントで投稿しないよう注意してください。

統計を見る

下タブ「統計」で、今月／今週／今日／通算の合計と、月別／曜日別／時間帯別の傾向グラフ、 「誰で／ジャンル」のランキングが見られます。

別の端末にデータを引き継ぐ

2つの方法があります。

方法1：引き継ぎコード（ふっかつのじゅもん）

  1. 元端末で「設定 → 引き継ぎコードを作る」 → コピー
  2. LINEで自分宛に送る／メモアプリに貼る／AirDrop などで新端末へ運ぶ
  3. 新端末で「設定 → 引き継ぎコードを取り込む」 → 貼り付け → 取り込む

方法2：JSONファイル

  1. 元端末で「設定 → JSONで書き出し」 → ダウンロードしたファイルを新端末に運ぶ
  2. 新端末で「設定 → JSONから読み込み」 → ファイルを選択

どちらの方法も既存データとマージされます（同じIDは上書き、新しいIDは追加）。

ホーム画面に追加（PWA）

iPhone（Safari）：画面下の共有ボタン → 「ホーム画面に追加」

Android（Chrome）：右上のメニュー → 「ホーム画面に追加」または「アプリをインストール」

データを完全に消したい

「設定 → 全データを削除」で、記録もお気に入りもすべて消去できます。取り消せません。
