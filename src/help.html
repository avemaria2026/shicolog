使い方

基本：記録する

  1. ホーム画面の大きな「＋1」ボタンを タップ
  2. 「誰で／何で」を入力 → 「保存」（スキップで未入力のまま記録もOK）
  3. 長押し（0.5秒） で、モーダルを開かずにメモなしで即記録もできます

※ プライバシーが気になる方は、設定画面から 4桁のPINロック を有効化できます（任意）。

履歴を編集・削除

  1. 下タブ「履歴」を開く
  2. 編集したい記録を タップ
  3. 内容を直して「保存」、または「削除」（確認ダイアログあり）

統計を見る

下タブ「統計」で、今月／今週／今日／通算の合計と、月別／曜日別／時間帯別の傾向グラフ、 「誰で／何で」のランキングが見られます。

PINロックを有効化する／PINを忘れたとき

本アプリは 初期状態ではPINロックなし で、誰でもすぐに使い始められます。 プライバシーが気になる場合は、設定画面の「PINロックを有効化」から4桁PINを設定してください。

一度PINを設定したあとに忘れてしまった場合、データは端末ローカルに保存されているため運営者側からの復旧はできません。 どうしても解除したい場合は、ブラウザのサイトデータをクリアすると初期状態に戻りますが、これまでの記録もすべて消えます。 大事な記録は事前にCSV／JSON書き出しでバックアップ しておくことを推奨します。

生体認証（指紋・顔）でロック解除

※ PINロックを有効化したあとに使えるオプションです。

  1. 下タブ「設定」を開く
  2. 「生体認証を有効化」をタップ → 端末の生体認証ダイアログで登録
  3. 以後、起動時に自動で指紋・顔認証ダイアログが出る（失敗時はPIN入力にフォールバック）

※ ご利用の端末・ブラウザが生体認証に対応している必要があります。

別の端末にデータを引き継ぐ

2つの方法があります。

方法1：引き継ぎコード（ふっかつのじゅもん）

  1. 元端末で「設定 → 引き継ぎコードを作る」 → コピー
  2. LINEで自分宛に送る／メモアプリに貼る／AirDrop などで新端末へ運ぶ
  3. 新端末で「設定 → 引き継ぎコードを取り込む」 → 貼り付け → 取り込む

方法2：JSONファイル

  1. 元端末で「設定 → JSONで書き出し」 → ダウンロードしたファイルを新端末に運ぶ
  2. 新端末で「設定 → JSONから読み込み」 → ファイルを選択

どちらの方法も既存データとマージされます（同じIDは上書き、新しいIDは追加）。

ホーム画面に追加（PWA）

iPhone（Safari）：画面下の共有ボタン → 「ホーム画面に追加」

Android（Chrome）：右上のメニュー → 「ホーム画面に追加」または「アプリをインストール」

データを完全に消したい

「設定 → 全データを削除」で、記録もPIN設定も生体認証情報もすべて消去できます。取り消せません。
